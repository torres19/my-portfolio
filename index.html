About

私について

普段は会社員として働きながら、スキマ時間で資格の勉強や、 シンプルなWebページづくり、小さな仕組みづくりを続けています。 自分や身近な人の「分かりづらい」「面倒くさい」を、 少しずつ軽くしていくのが好きです。

このポートフォリオでは、肩書きよりも 「どんなことを学び」「どんな形で残しているか」を中心に載せています。

Output

つくれるもの・やっていること

本業とは切り離した個人の範囲で、こんなアウトプットができます。 ガチ営業ではなく、「このくらいまでは一人で組めます」の目安として。

静的サイト構築

HTML / CSS / バニラJSで完結する軽いWebページ。 GitHub Pagesなどを使ったポートフォリオ・LPの構成調整。

学習用コンテンツ

資格テキストの要点整理、確認用クイズページなど、 「勉強した内容を残す」ためのシンプルなコンテンツづくり。

情報整理・ちいさな仕組み

メモテンプレート、チェックリスト、簡単な入力フォームなど、 日常や仕事を少し楽にする仕組みの試作。

ローカルの紹介ページ

応援したいお店や場所の紹介ページ制作。 多言語対応や写真レイアウトの調整など。

Study

勉強していること

学んだことをそのまま流してしまうのがもったいないので、 自分用の整理も兼ねて、要点メモやクイズとして残していきます。

  • 資格系：簿記 / マネー系 / IT系 などの基礎整理
  • Web基礎：HTML / CSS / バニラJS の実験
  • 身近な仕事に役立つ知識のメモ化

※ ここから各資格ごとのテキスト・小テストページへリンク予定。 ログインや記録は行わず、気軽に使える形を想定しています。

Hobby

好きなこと

生活の温度を保ってくれているものたち。 深堀りしすぎない程度に、断片的に置いていきます。

  • 植物：育てたり眺めたり、増えたり減ったり。
  • 料理：気分で作るごはんと、お酒に合うもの。
  • スポーツや格闘技を見る時間。
  • ローカルな居酒屋やお店をふらっと巡ること。
